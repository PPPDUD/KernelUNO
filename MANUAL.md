# KernelUNO Manual
## File management
### ls command
The `ls` command lists all files and folders in the current directory.

### cd command
The `cd` command changes the current directory to the first argument given.

### cat command
The `cat` command reads the file specified in the first argument.

### df and free commands
The `df` and `free` commands tell you the amount of free memory.

### rm command
The `rm` command removes the specified file.
