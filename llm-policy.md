Title: Policy on the use of Large Language Models (LLMs)


General Rule: The use of LLMs (such as ChatGPT, Claude, Copilot) to generate code for KernelUNO is strictly prohibited. All contributions must be written by humans who fully understand the logic they are submitting.


Rationale: Kernel code requires extreme precision and deep understanding of side effects. LLM-generated code often contains subtle bugs, hallucinations, or violates specific architectural patterns of this project.


Enforcement:Submissions suspected of being LLM-generated will be subjected to additional scrutiny.


If a contributor cannot explain the technical decisions behind their code, the PR will be closed.


Repeated violations may lead to a temporary or permanent ban on contributing.

This file was created in response to the recent increase in pull requests featuring AI-generated code.
