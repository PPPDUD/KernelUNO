# Policy on the use of Large Language Models (LLMs)
The use of LLMs (such as ChatGPT, Claude, Copilot) to generate code for KernelUNO is strictly prohibited. All contributions must be written by humans who fully understand the logic they are submitting, because mistakes in low-level C++ code (as may be found in this project) can cause massive problems for the end-user.

Any pull requests that obviously contain LLM-generated code in them will not be merged.

Repeated violations may lead to a temporary or permanent ban on contributing.
