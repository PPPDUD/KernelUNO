# Coconix
Coconix is a Unix-like shell for Arduino Uno boards.

## Commands
- `uname`
- `cd`
- `ls`
- `pwd`
- `mkdir`
- `touch`
- `cat`
- `echo`
- `rm`
- `info`
- `pinmode`
- `write`
- `read`
- `gpio`
- `sh`
- `uptime`
- `dmesg`
- `df`
- `free`
- `whoami`
- `clear`
- `reboot`
- `pwm`
