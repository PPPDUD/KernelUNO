# KernelUNO v1.0 - a Ram filesystem emulator and simple shell for your arduino UNO R3 

# Features

- Unix-like Shell with 23 commands

- GPIO support

# Commands
- uname
- cd
- ls
- pwd
- mkdir
- touch
- cat
- echo
- rm
- info
- pinmode
- write
- read
- gpio
- sh
- uptime
- dmesg
- df
- free
- whoami
- clear
- reboot
